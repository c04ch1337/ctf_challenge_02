# CTF Challenge 2: The Hacker's Lair

## Story
Deep in the dark web lies "The Hacker's Lair," a server used by cybercriminals to plan their attacks. Your mission is to infiltrate the server, exploit vulnerabilities in their web application, and retrieve hidden flags. Each flag corresponds to an OWASP Top 10 vulnerability, so you'll learn as you play.

## Objective
Find all 10 hidden flags by exploiting vulnerabilities in the web application. Each flag corresponds to an OWASP Top 10 vulnerability.

## How to Play
1. Start the challenge by running `./start.sh`.
2. Access the web application at `http://localhost:8080/juice-shop`.
3. Use your skills to find and exploit vulnerabilities.
4. Submit the flags to complete the challenge.

## Hints
If you're stuck, check the `hints.md` file for guidance.

Good luck, investigator!
